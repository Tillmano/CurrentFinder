Example Circuits

To input a circuit press the 'add battery and 'add resisitor' buttons

Input the ID in order of components you add, starting at 1

Nodes that a component is between can be thought of as sections of wire, where components meet.

Input for this circuit:

Component 1 is a 1.5 volt battery between node 1 and 2.

Component 2 is a 3 ohm resistor between node 2 and 1.

Input for this circuit:

Component 1 is a 9 volt battery between node 2 and 3.

Component 2 is a 3 ohm resistor between node 1 and 3.

Component 3 is a 2 ohm resistor between node 2 and 1.

Component 4 is a 1 ohm resistor between node 2 and 1.

Component 3 and 4 are connected in parallel.
